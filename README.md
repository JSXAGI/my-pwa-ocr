# 🔍 安心・安全 OCRエディタ

スマホやPCのブラウザで動く、プライバシー重視型の文字起こし（OCR）アプリです。

## 🌟 このアプリのすごいところ

1. **完全プライバシー保護（サーバー送信なし）**
   - 画像の解析はすべて、あなたのスマホやPCの中（ブラウザ上）だけで行われます。
   - インターネット上のサーバーに画像をアップロードしないため、機密書類や個人情報も安心して読み取れます。

2. **インストール不要**
   - URLを開くだけですぐに使えます。
   - 「ホーム画面に追加」をすれば、まるで普通のアプリのように全画面で起動します。

3. **オフライン対応**
   - 一度読み込めば、デバイス内だけで動作する仕組み（Tesseract.js）を採用しています。

## 📸 使い方

1. 画面中央の「📸 クリックして撮影」をタップします。
2. カメラが起動するので、文字を真っ直ぐ、ハッキリと撮影します。
3. 数秒〜数十秒待つと（※初回は少し時間がかかります）、下のテキストエリアに文字が抽出されます。
4. 「結果をコピー」ボタンで、メモ帳やメールに貼り付けられます。

## 💡 きれいに読み取るコツ

- **明るい場所で撮る**: 暗いと誤字が増えます。
- **真っ直ぐ撮る**: 斜めになっているとAIが迷ってしまいます。
- **文字を大きく**: 遠すぎず、文字がハッキリ映る距離で。
